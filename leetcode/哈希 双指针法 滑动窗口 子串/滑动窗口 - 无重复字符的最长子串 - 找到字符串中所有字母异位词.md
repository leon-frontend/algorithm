### _**无重复字符的最长子串**_

> _**问题描述**_
>
> > [!important] **[LeetCode - 无重复字符的最长子串](https://leetcode.cn/problems/longest-substring-without-repeating-characters/)**
>
> - 给定一个字符串 `s` ，请你找出其中不含有重复字符的 **最长子串** 的长度。

> _**使用滑动窗口**_
>
> 1. 使用两个指针 `**left**` 和 `**right**` 来构建一个滑动窗口，初始时两个指针都指向字符串的开头。
> 
> 1. 遍历字符串，移动 `**right**` 指针，并将遍历到的字符添加到集合中，表示当前窗口中包含的字符。
> 
> 1. 如果遇到重复的字符，移动 `**left**` 指针直到重复字符不再在集合中。
> 
> 1. 在遍历的过程中，不断更新最长子串的长度。
> 
> ```JavaScript
> /**
>  * @param {string} s
>  * @return {number}
>  */
> var lengthOfLongestSubstring = function(s) {
>   // 使用 Set 判断子串中是否有重复元素
>   const set = new Set()
>   // 使用双指针表示一个滑动窗口
>   let left = 0, right = 0; maxLen = 0  
>   while (right < s.length) {
>     // 获取滑动窗口最右侧的字符
>     const char = s[right]
>     if (!set.has(char)) {
>       // 若 char 不在 set 中，则添加进 set
>       set.add(char) 
>       // 更新 maxLen
>       maxLen = Math.max(maxLen, right - left + 1)
>       // 只有当 char 不在 set 中时，right 向右移动
>       right++
>     } else {
>       // 若 char 在 set 中，则移动 left 指针直到重复字符不再在集合中
>       set.delete(s[left])
>       left++
>     }
>   }
>   return maxLen
> };
> ```

### _**找到字符串中所有字母异位词**_

> _**问题描述**_
>
> > [!important] **[LeetCode - 找到字符串中所有字母异位词](https://leetcode.cn/problems/find-all-anagrams-in-a-string/description/?envType=study-plan-v2&envId=top-100-liked)**
>
> - 给定两个字符串 `s` 和 `p`，找到 `s` 中所有 `p` 的 **异位词** 的子串，返回这些子串的起始索引。不考虑答案输出的顺序。**异位词**指由相同字母重排列形成的字符串（包括相同的字符串）。

> **使用滑动窗口和双指针解题**
>
> - 用 **26** 位数组记录每个字母出现的次数 ⇒ 创造一个滑动窗口数组和一个字符串 **p** 数组
> 
> - 用双指针固定滑动窗口的大小
> 
> - _==**重点 ⇒**==_ 如何维护滑动窗口的数组；如何设置控制指针的条件
> 
> ```Java
> /**
>  * @param {string} s
>  * @param {string} p
>  * @return {number[]}
>  */
> var findAnagrams = function(s, p) {
>   // 声明结果数组
>   const res = []
>   // 声明一个数组，记录滑动窗口中每个字符出现的次数
>   const window = Array(26).fill(0)
>   // 声明一个数组，记录 p 中每个字符出现的次数
>   const pCount = Array(26).fill(0)
>   // 遍历字符串 p，更新 pCount
>   // "ABC".charCodeAt(0); // 返回 65，获取"ABC"中索引为0的字符的 ASCII 码
>   for (let char of p) pCount[char.charCodeAt() - 'a'.charCodeAt()]++
>   // 定义滑动窗口的左边界
>   let left = 0
>   // 遍历 s 中的每个字符，right 代表滑动窗口的右边界
>   for (let right = 0; right < s.length; right++) {
>     // 利用 当前字符 更新 window 数组
>     window[s[right].charCodeAt() - 'a'.charCodeAt()]++
>     // 若 滑动窗口的长度 等于 p 的长度，则比较 window 和 pCount 是否相等
>     if (right - left + 1 === p.length) {
>       // 若 window 和 pCount 相等，记录解
>       if (window.every((char, index) => char === pCount[index])) res.push(left)
>       // 无论 window 和 pCount 是否相等，都要先更新 window 数组，再移动 left 指针
>       // 滑动窗口的大小只有和 p 的长度相等时，才可能找到解
>       window[s[left].charCodeAt() - 'a'.charCodeAt()]--
>       left++
>     }
>   }
>   return res
> };
> ```
