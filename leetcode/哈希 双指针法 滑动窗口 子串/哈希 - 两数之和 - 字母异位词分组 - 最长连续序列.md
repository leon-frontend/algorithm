### _**两数之和**_

> _**问题描述**_
>
> > [!important] **[LeetCode - 两数之和](https://leetcode.cn/problems/two-sum/description/?envType=study-plan-v2&envId=top-100-liked)**
>
> - 给定一个整数数组 **nums** 和一个整数目标值 **target**，请你在该数组中找出和为目标值 **target** 的那两个整数，并返回它们的数组下标。你可以假设每种输入只会对应一个答案。但是，数组中同一个元素在答案里不能重复出现。你可以按任意顺序返回答案。
> 
> ```Java
> /**
>  * @param {number[]} nums
>  * @param {number} target
>  * @return {number[]}
>  */
> // 使用 map 映射求解
> var twoSum = function(nums, target) {
>   // 使用 对象 => 映射元素下标和元素值的关系 value: index
>   const map = {}
>   // 初始化 map
>   map[nums[0]] = 0
>   for (let i = 1; i < nums.length; i++) {
>     let another = target - nums[i]
>     // map[another] 会在 map 对象中检查键 another 对应的值。
>     // 如果该值为 0，if (map[another]) 会被错误地评估为 false，因为 0 是一个 false 值。
>     // if (map[another]) return [map[another], i]
>     // 应该使用 in 操作符
>     if (another in map) return [map[another], i]
>     map[nums[i]] = i
>   }
> };
> ```

### _**字母异位词分组**_

> _**问题描述**_
>
> > [!important] **[LeetCode - 字母异位词分组](https://leetcode.cn/problems/group-anagrams/?envType=study-plan-v2&envId=top-100-liked)**
>
> - 给你一个字符串数组，请你将**字母异位词**组合在一起。可以按任意顺序返回结果列表。**字母异位词**是由重新排列源单词的所有字母得到的一个新单词。
> 
> ```JavaScript
> /**
>  * @param {string[]} strs
>  * @return {string[][]}
>  */
>  // 使用 map 映射求解
> var groupAnagrams = function(strs) {
>   // 创建一个 Map，映射 strs 中的排序后的每个元素 和 对应的字母异位词数组
> 	// new 不能省略
>   const map = new Map()
>   // 使用 forEach 遍历 strs 中的每个元素
>   strs.forEach(str => {
>     // 将每个元素（字符串）转换成 strArr 数组 => 对 strArr 数组进行排序 => 将排序后的数组转换为字符串
>     const keyStr = str.split('').sort().join()
>     // 判断 map 中是否有值为 newStr 的 key
>     if (map.has(keyStr)) map.get(keyStr).push(str)
>     else map.set(keyStr, [str])
>   })
>   return Array.from(map.values())
> };
> ```

### _**最长连续序列**_

> _**问题描述**_
>
> > [!important] **[LeetCode - 最长连续序列](https://leetcode.cn/problems/longest-consecutive-sequence/)**
>
> - 给定一个未排序的整数数组 `nums` ，找出数字连续的最长序列（不要求序列元素在原数组中连续）的长度。请你设计并实现时间复杂度为 `O(n)` 的算法解决此问题。
> 

> **算法分析**
>
> > [!important] **[LeetCode - 查看图解，一目了然](https://leetcode.cn/problems/longest-consecutive-sequence/solutions/276931/zui-chang-lian-xu-xu-lie-by-leetcode-solution/?envType=study-plan-v2&envId=top-100-liked)**
>
> 1. 创建一个空的哈希表 `**numSet**`，用于存储数组中的每个数字
> 
> 2. 遍历数组 `nums`，将每个数字存入哈希表中
> 
> 3. 对于每个数字 `num`，检查其左相邻的数字是否在哈希表中
>     
>     1. 如果不存在，则`while (numSet.has(numTem + 1))`，直到找不到`numTem + 1`
>     
>     2. 如果存在，则跳过，避免冗余循环。
>         
>         1. 若 **num** 为 **2**，若 **1** 在 **set** 中，应该从 **1** 开始遍历，不用从 **2** 开始遍历
>         
>     
> 
> 4. 更新最长序列的长度
> 
> 5. 最终返回最长序列的长度
> 
> ```Java
> /**
>  * @param {number[]} nums
>  * @return {number}
>  */
> var longestConsecutive = function(nums) {
>   // 使用 set 过滤 nums 中的重复元素，避免冗余循环
>   const set = new Set(nums)
> 
>   let longestLen = 0
> 
>   // 遍历 set 中的每个元素
>   for (let num of set) {
>     // 检查其 左相邻 的数字是否在 Set 中
>     // 若存在，则直接跳过，避免冗余循环
>     if (!set.has(num - 1)) {
>       // 记录当前 num 的连续长度
>       let curLen = 1
>       // 由于不能修改数组数据，所以声明 curNum 用于 ++ 操作
>       let curNum = num
> 
>       while (set.has(curNum + 1)) {
>         curLen++
>         curNum++
>       }
> 
>       // 更新最大连续长度
>       longestLen = Math.max(longestLen, curLen)
>     }
>   }
> 
>   return longestLen
> };
> ```
