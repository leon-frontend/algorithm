### _移动零_

> _**问题描述**_
>
> > [!important] **[LeetCode - 移动零](https://leetcode.cn/problems/move-zeroes/)**
>
> - 给定一个数组 `nums`，编写一个函数将所有 `0` 移动到数组的末尾，同时保持非零元素的相对顺序。请注意必须在不复制数组的情况下原地对数组进行操作。
> 
> ```Java
> /**
>  * @param {number[]} nums
>  * @return {void} Do not return anything, modify nums in-place instead.
>  */
> var moveZeroes = function(nums) {
>   // 使用双指针
>   // 一个指针遍历数组 => i
>   // 一个指针指向非零元素应该在的位置 => nonZero
>   let nonZero = 0
>   for (let i = 0; i < nums.length; i++) {
>     //  只有当 nums[i] 不为 0 时，nonZero++
>     if (nums[i] !== 0) {
>       [nums[i], nums[nonZero]] = [nums[nonZero], nums[i]]
>       nonZero++
>     }
>   }
> };
> 
> // ------------------- 解法 2 -------------------
> public void moveZeroes(int[] nums) {
>     int len = nums.length;
>     int j = 0;		
> 		// 将全部非 0 元素复制到数组前面
>     for (int i = 0; i < len; i++) {
>         if (nums[i] != 0) {
>             nums[j++] = nums[i];
>         }    
>     }    
> 		// 非0元素统计完了，剩下的都是0了
> 		// 所以第二次遍历把末尾的元素都赋为 0 即可
>     for (int i = j; i < len; i++) nums[i] = 0;
> }
> ```

### _盛最多水的容器_

> _**问题描述**_
>
> > [!important] **[LeetCode - 盛最多水的容器](https://leetcode.cn/problems/container-with-most-water/)**
>
> - 给定一个长度为 `n` 的整数数组 `height` 。有 `n` 条垂线，第 `i` 条线的两个端点是 `(i, 0)` 和 `(i, height[i])` 。找出其中的两条线，使得它们与 `x` 轴共同构成的容器可以容纳最多的水。返回容器可以储存的最大水量。注意你不能倾斜容器。
> 
> ![[双指针法 - 1.png|Untitled 63.png]]
>
> ```Java
> /**
>  * @param {number[]} height
>  * @return {number}
>  */
>  // 使用双指针
> var maxArea = function(height) {
>   // 存储的水量大小，取决于 较矮 的垂线
>   // 若移动 较高 的垂线，存水量要么不变，要么变小
>   // 若移动 较矮 的垂线，存水量才有可能变大
> 
>   // 另外，只要垂线向中间移动，横向距离就会变短
>   // 所以移动 较高 的垂线，存水量必变小
>   // 因此只有移动较矮的垂线，存水量才可能变大
>   let left = 0, right = height.length - 1
>   let maxSize = 0
>   while (left < right) {
>     // if (height[left] < height[right]) {
>     //   maxSize = Math.max(maxSize, (right - left) * height[left])
>     //   left++ // 移动短板
>     // }else {
>     //   maxSize = Math.max(maxSize, (right - left) * height[right])
>     //   right-- // 移动短板
>     // }
>     maxSize = height[left] < height[right] ? 
>               Math.max(maxSize, (right - left) * height[left++]) :
>               Math.max(maxSize, (right - left) * height[right--])
>   }
>   return maxSize
> };
> ```

### _**三数之和**_

> _**问题描述**_
>
> > [!important] **[LeetCode - 三数之和](https://leetcode.cn/problems/3sum/)**
>
> - 给你一个整数数组 `nums` ，判断是否存在三元组 `[nums[i], nums[j], nums[k]]` 满足 `i != j`、`i != k` 且 `j != k` ，同时还满足 `nums[i] + nums[j] + nums[k] == 0` 。请你返回所有和为 `0` 且不重复的三元组。注意答案中不可以包含重复的三元组
> 
> ```Java
> /**
>  * @param {number[]} nums
>  * @return {number[][]}
>  */
> var threeSum = function(nums) {
>   // 对于排好序的数组，要求当前的元素值和上一个不同 => 答案中不可以包含重复的三元组
>   const res = []
>   const len = nums.length
> 
>   // 给数组排序，升序，最好加上(a, b) => a - b
> 	// 如果没有提供 compareFn，所有非 undefined 的数组元素都会被转换为字符串，
> 	// 并按照 UTF-16 码元顺序比较字符串进行排序
>   nums.sort((a, b) => a - b)
> 
>   // 遍历数组中的每个元素
>   for (let first = 0; first < len; first++) {
>     // 剪枝 1
>     // 对于升序数组，若前三个最小的数加起来大于0，那么后面不论是什么数加起来都会大于0
>     if (first + 2 < len && nums[first] + nums[first + 1] + nums[first + 2] > 0) break
> 
>     // 剪枝 2
>     // 若第一个数加上最后两个最大的数是小于0 的，那么可以直接跳过这层循环
>     if (nums[first] + nums[len - 1] + nums[len - 2] < 0) continue
> 
>     // 剪枝 3
>     // 对于排好序的数组，要求当前的元素值和上一个不同，否则会产生重复的三元组
>     if (first > 0 && nums[first] === nums[first - 1]) continue
> 
>     // third 从后往前遍历数组
>     let third = len - 1
>     const target = -nums[first]
> 
>     for (let second = first + 1; second < len; second++) {
>       // 剪枝 4
>       // 对于排好序的数组，要求当前的元素值和上一个不同，否则会产生重复的三元组
>       if (second > first + 1 && nums[second] === nums[second - 1]) continue
> 
>       // 要求 second < third,且 third 指向的是最大的数
>       // 所以第一次循环中 nums[second] + nums[third] 是最大值
>       while (second < third && nums[second] + nums[third] > target) third--
> 
>       // second === third 时，说明无法找到满足条件的 second 和 third
>       if (second === third) break
> 
>       if (nums[second] + nums[third] === target) {
>         // 创建一个数组，保存 first / second / third 指针对应的值
>         res.push([nums[first], nums[second], nums[third]])
>       }
>     }
>   }
> 
>   return res
> };
> ```

### _**接雨水**_

> _**问题描述**_
>
> > [!important] **[LeetCode - 接雨水](https://leetcode.cn/problems/trapping-rain-water/)**
>
> - 给定 `n` 个非负整数表示每个宽度为 `1` 的柱子的高度图，计算按此排列的柱子，下雨之后能接多少雨水。
> 
> ![[双指针法 - 3.png|Untitled 2 35.png]]

> _**解法 1 ⇒ 使用动态规划优化按列求解**_
>
> - 时间复杂度 **O(n)**，空间复杂度 **O(n)** ，用来保存每一列左边最高的墙和右边最高的墙。
> 
> ```Java
> // 使用 动态规划 优化按列求解算法
> // 根据当前列 i ，找出左边最高的列和右边最高的列，使用动态规划在此处做优化
> // 当前列是否能接水，取决于当前列 i 的高度是否小于 min(左边最高列，右边最高列)
> class Solution {
>   public int trap(int[] height) {
>     int len = height.length;
>     int sum = 0;
>     // 使用数组记录当前列的左右两边的最高的列，减少一次循环
>     int[] max_left = new int[len];
>     int[] max_right = new int[len];        
>     // 最左最右两边的列不可能接到雨水，所以当前列 i 的取值范围是 1 ~ len - 1
>     // 使用动态规划，找出当前列 i 左边最高的列
>     for (int i = 1; i < len - 1; i++) {
>       max_left[i] = Math.max(max_left[i - 1], height[i - 1]);
>     }
>     // 使用动态规划，找出当前列 i 右边最高的列
>     for (int i = len - 2; i > 0; i--) {
>       max_right[i] = Math.max(max_right[i + 1], height[i + 1]);
>     }
>     for (int i = 1; i < len - 1; i++) {
>       int min = Math.min(max_left[i], max_right[i]);
>       if (height[i] < min) sum += (min - height[i]);
>     }
>     return sum;
>   }
> }
> ```

> _**使用 双指针法，优化以上算法**_
>
> 1. 双指针同时开两个柱子接水
> 
> 1. 对于每一个柱子接的水，那么它能接的`**水 = min(左右两边最高柱子）- 当前柱子高度**`
> 
> 1. 同样的，两根柱子要一起求接水，同样要知道它们左右两边最大值的较小值。
> 
> 1. 假设两柱子分别为`**i，j(代表下标)**`。那么就有`**iLeftMax, iRightMax, jLeftMax, jRightMax**` 这 4 个变量。
> 
> 1. 假设==`**j>i**`== ，故 ==`**jLeftMax >= iLeftMax，iRigthMax >= jRightMax.**`==
> 
> 1. 如果`**iLeftMax > jRightMax**`，则必有`**jLeftMax >= jRightMax**`，所有能接 _**j**_ 点的水。
> 
> 1. 如果==`**jRightMax > iLeftMax**`==，则必有==`**iRightMax >= iLeftMax**`==，所以能接 _**i**_ 点的水。
> 
> 1. 而上面我们实际上只用到了`**iLeftMax，jRightMax**`两个变量，故我们维护这两个即可。
> 
> 1. 即要比较 _**i 的左边的最高柱子和 j 右边的最高柱子大小**_ 。
> 
> ```Java
> // 双指针法
> // 使用指针代替动态规划中的数组
> class Solution {
>     public int trap(int[] height) {
>         int len = height.length;
> 
>         // 指向左边要接水的柱子, 最左边的柱子不可能接到水
>         int left = 1; 
>         // 指向右边要接水的柱子，最右边的柱子不可能接到水
>         int right = len - 2; 
> 
>         int maxLeft = 0;
>         int maxRight = 0;
>         int sum = 0;
> 
>         // 循环至两个指针相遇，遍历一次所有的柱子
>         for (int i = 1; i < height.length - 1; i++) {
>             maxLeft = Math.max(maxLeft, height[left - 1]); // 更新 maxLeft
>             maxRight = Math.max(maxRight, height[right + 1]);
>             // 使用左边的指针遍历
>             if (maxLeft < maxRight) {
>                 // 此时的 maxLeft 肯定是 小于 maxRight 的
>                 if (maxLeft > height[left]) sum += (maxLeft - height[left]);
>                 left++; // 继续处理下一个柱子
>             } else { // 使用右边的指针遍历
>                 // 此时的 maxRight 肯定是 小于 maxLeft 的
>                 if (maxRight > height[right]) sum += (maxRight - height[right]);
>                 right--; // 继续处理下一个柱子
>             }             
>         }
>         return sum;
>     }
> }
> ```
