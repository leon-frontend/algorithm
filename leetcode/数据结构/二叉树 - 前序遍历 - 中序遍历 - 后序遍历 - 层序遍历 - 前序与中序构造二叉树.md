### _**前序遍历**_

> _**问题描述**_
>
> > [!important] **[LeetCode - 前序遍历](https://leetcode.cn/problems/binary-tree-preorder-traversal/description/)**
>
> - 给你二叉树的根节点 `root` ，返回它节点值的 **前序** 遍历。

> _**解法 1 ⇒ 递归**_
>
> - 时间复杂度：$O(n)$，其中 $n$ 是二叉树的节点数。每一个节点恰好被遍历一次。
> 
> - 空间复杂度：$O(n)$，为递归过程中栈的开销，平均情况下为 $O(log⁡n)$，最坏情况下树呈现链状，为 $O(n)$。
> 
> ```Java
> public List<Integer> preorderTraversal(TreeNode root) {
>   List<Integer> res = new ArrayList<>();
>   preOrder(root, res);
>   return res;
> }
> 
> public void preOrder(TreeNode root, List<Integer> res) {
>   if (root == null) return;
>   res.add(root.val);
>   preOrder(root.left, res);
>   preOrder(root.right, res);
> }
> ```

> _**解法 2 ⇒ 栈**_
>
> ```Java
> public List<Integer> preorderTraversal(TreeNode root) {
>   List<Integer> res = new ArrayList<>();
>   if (root == null) return res;  
>   Deque<TreeNode> stack = new LinkedList<>();
>   TreeNode node = root;  
>   while (node != null || !stack.isEmpty()) {
>     while (node != null) {
>       res.add(node.val);
>       stack.push(node);
>       node = node.left;
>     }
>     node = stack.pop();
>     node = node.right;
>   }
>   return res;
> }
> ```

> _**解法 3 ⇒ Morris 遍历**_
>
> > [!important] **[知乎 - 神级遍历——morris](https://zhuanlan.zhihu.com/p/101321696)**
>
> - ==**Morris**== ==的整体思路就是将以某个根结点开始，找到它== ==左子树的最右侧节点== ==之后与这个== ==根结点====进行连接。==每个节点会到达两次，第一次访问节点时，存储节点值。和中序遍历相比，将 **x** 的值加入答案数组的代码位置不同。
> 
> - 假设当前遍历到的节点为 _**x**_ , 每次处理以 _**x**_ 为根节点的子树
> 
> - 如果 _**x**_ 无左孩子，先将 _**x**_ 的值加入答案数组，再访问 _**x**_ 的右孩子，即`**x=x.right**`
> 
> - 如果 _**x**_ 有左孩子，则找到 _**x**_ 左子树上最右的节点（即左子树中序遍历的最后一个节点，_**x**_ 在中序遍历中的前驱节点），我们记为 _**pre**_ 。根据 _**pre**_ 的右孩子是否为空，进行如下操作
>     
>     - 如果 _**pre**_ 的右孩子为空，则将其右孩子指向 _**x**_，==将== _==**x**==_ ==的值加入答案数组==，然后访问 _**x**_ 的左孩子，即 ==`**x=x.left**`==
>     
>     - 如果 **pre** 的右孩子不为空，则此时其右孩子指向 **x**，说明==已经遍历完== ==**x**== ==的左子树==
>         
>         - 将 **pre** 的右孩子置空，然后访问 **x** 的右孩子，即 `**x = x.right**`
>         
>     
> 
> ```Java
> public List<Integer> preorderTraversal(TreeNode root) {
>     List<Integer> res = new ArrayList<>();
>     if (root == null) return res;    
>     TreeNode pre = null;
>     while (root != null) {
>         if (root.left != null) {
>             // 先找到 pre 节点
>             pre = root.left;
>             while (pre.right != null && pre.right != root) pre = pre.right;
>             // 判断 pre.right 是否指向 root
>             if (pre.right == null) {
>                 res.add(root.val);
>                 pre.right = root;
>                 root = root.left;
>             }else {
>                 // 若 pre.right 指向 root，则是第二次经过该节点，左子树遍历完毕
>                 pre.right = null;
>                 root = root.right;
>             }
>         }else {
>             res.add(root.val); 
>             root = root.right;
>         }
>     }
>     return res;        
> }
> ```

### _**中序遍历**_

> _**问题描述**_
>
> > [!important] **[LeetCode - 二叉树的中序遍历](https://leetcode.cn/problems/binary-tree-inorder-traversal/description/?envType=study-plan-v2&envId=top-100-liked)**
>
> - 给定一个二叉树的根节点 `root` ，返回 _它的 **中序** 遍历_ 。

> _**解法 1 ⇒ 递归**_
>
> - 时间复杂度 $O(n)$，其中 _**n**_ 为二叉树节点的个数。二叉树的遍历中每个节点会被访问一次且只会被访问一次
> 
> - 空间复杂度 $O(n)$，空间复杂度取决于递归的栈深度，而栈深度在二叉树为一条链的情况下会达到 $O(n)$ 的级别。
> 
> ```Java
> /**
>  * Definition for a binary tree node.
>  * function TreeNode(val, left, right) {
>  *     this.val = (val===undefined ? 0 : val)
>  *     this.left = (left===undefined ? null : left)
>  *     this.right = (right===undefined ? null : right)
>  * }
>  */
> /**
>  * @param {TreeNode} root
>  * @return {number[]}
>  */
> var inorderTraversal = function(root) {
>   const inOrder = (root, res) => {
>     if (!root) return 
>     inOrder(root.left, res)
>     res.push(root.val)
>     inOrder(root.right, res)
>   }
>   const res = []
>   inOrder(root, res)
>   return res
> };
> ```

> _**解法 2 ⇒ 栈**_
>
> - 时间复杂度：$O(n)$，其中 _**n**_ 为二叉树节点的个数。二叉树的遍历中每个节点会被访问一次且只会被访问一次
> 
> - 空间复杂度：$O(n)$。空间复杂度取决于栈深度，而栈深度在二叉树为一条链的情况下会达到 $O(n)$ 的级别
> 
> ```SQL
> public List<Integer> inorderTraversal(TreeNode root) {
>     List<Integer> res = new ArrayList<Integer>();
>     Deque<TreeNode> stk = new LinkedList<TreeNode>();
>     while (root != null || !stk.isEmpty()) {
> 				// 访问左子树
>         while (root != null) {
>             stk.push(root);
>             root = root.left;
>         }
>         root = stk.pop();
>         res.add(root.val);
>         root = root.right;
>     }
>     return res;
> }
> ```

>

### _后序遍历_

> _**问题描述**_
>
> > [!important] **[LeetCode - 后序遍历](https://leetcode.cn/problems/binary-tree-postorder-traversal/)**
>
> - 给你一棵二叉树的根节点 `root` ，返回其节点值的 **后序遍历** 。

> _**解法 1 ⇒ 递归**_
>
> ```Java
> public List<Integer> postorderTraversal(TreeNode root) {
>     List<Integer> res = new ArrayList<>();
>     postOrder(root, res);
>     return res;
> }
> public void postOrder(TreeNode root, List<Integer> res) {
>     if (root == null) return;
>     postOrder(root.left,res);
>     postOrder(root.right, res);
>     res.add(root.val);
> }
> ```

> _**解法 2 ⇒ 双栈**_
>
> > [!important] **[知乎 - 二叉树后序遍历的两种易写的非递归写法](https://zhuanlan.zhihu.com/p/80578741)**
>
> - 前序遍历的遍历顺序是 根左右，后序遍历的遍历顺序是 左右根
> 
> - 先求出遍历顺序是 根右左 的节点序列，再倒序，便刚好是后序遍历的顺序 左右根
> 
> ```Java
> public List<Integer> postorderTraversal(TreeNode root) {
>     List<Integer> res = new ArrayList<>();    
>     // 按照根右左的顺序遍历
>     Deque<TreeNode> stackTraversal = new LinkedList<>();
>     // 保存节点序列的栈
>     Deque<TreeNode> stackStore = new LinkedList<>();
>     while(root != null || !stackTraversal.isEmpty()) {
>         while (root != null) {
>             stackTraversal.push(root);
>             stackStore.push(root);
>             root = root.right;
>         }
>         root = stackTraversal.pop();
>         root = root.left;
>     }    
>     while (!stackStore.isEmpty()) res.add(stackStore.pop().val);
>     return res;
> }
> ```

> _**解法 3 ⇒ 单栈**_
>
> > [!important] **[知乎 - 二叉树后序遍历的两种易写的非递归写法](https://zhuanlan.zhihu.com/p/80578741)**
>
> - 在后序遍历中，访问完左孩子，会经过根节点，但是我们不会去访问它，而是会选择先访问它的右孩子。所以在这种情况下，我们会将根节点留在栈中不弹出，等到需要访问它的时候再出。
> 
> - _**什么时候才需要访问根节点呢？⇒**_ ==访问完右子节点之后==。记录一下上一次访问的节点，然后判断一下当前经过的节点和上一次访问的节点是什么关系。如果当前经过的节点的右子节点是上一次访问的节点，显然我们需要访问当前节点了
> 
> - 总结起来，我们什么时候才能访问节点。有如下两种情况：_**1.**_ 当前经过节点是 叶子节点；_**2.**_ 当前经过节点的右子节点是上一次访问的节点
> 
> ```Java
> public List<Integer> postorderTraversal(TreeNode root) {
>     List<Integer> res = new ArrayList<>();    
>     Deque<TreeNode> stack = new LinkedList<>();    
>     TreeNode pre = null; // 记录上一次访问的节点
>     while (root != null || !stack.isEmpty()) {
>         while (root != null) {
>             stack.push(root);
>             root = root.left;
>         }
>         // 此时的 root 没有左孩子
>         root = stack.pop();
>         if (root.right == null || root.right == pre) {
>             res.add(root.val);
>             pre = root; // 更新 pre
>             // 此处为了跳过下一次循环的访问左子节点的过程，直接进入栈的弹出阶段
>             // 因为但凡在栈中的节点，它们的左子节点都肯定被经过且已放入栈中
>             // 也起着推进递归进行的作用
>             root = null;
>         }else {
>             // 有未访问的右孩子
>             stack.push(root);
>             root = root.right;
>         }
>     }
>     return res;
> }
> ```

### _**层序遍历**_

> _**问题描述**_
>
> > [!important]
> >
> > **[LeetCode - 层序遍历](https://leetcode.cn/problems/binary-tree-level-order-traversal/description/?envType=study-plan-v2&envId=top-100-liked)**
>
> - 给你二叉树的根节点 `root` ，返回其节点值的 **层序遍历** 。 即逐层地，从左到右访问所有节点。

> _**广度优先遍历**_
>
> - 在每次循环时，判断队列大小，保证仅处理当前层的节点
> 
> - 时间复杂度：每个点进队出队各一次，故渐进时间复杂度为 $O(n)$
> 
> - 空间复杂度：队列中元素的个数不超过 _**n**_ 个，故渐进空间复杂度为 $O(n)$
> 
> ```Java
> var levelOrder = function(root) {
>   // 声明返回结果
>   const res = []
>   if (!root) return res
>   // 使用队列进行广度优先遍历，并初始化
>   const queue = [root]
>   while (queue.length) {
>     // 记录每一层的节点数量
>     let curLevelSize = queue.length
>     // 声明一个数组，保存这一层节点的信息
>     let curLevelNode = []
>     for (let i = 0; i < curLevelSize; i++) {
>       // 将元素出队，并添加至 curLevelNode 数组中保存节点
>       let node = queue.shift()
>       curLevelNode.push(node.val)      
>       // 将 node 的左右孩子入队
>       if (node.left) queue.push(node.left)
>       if (node.right) queue.push(node.right)
>     }
>     // 处理完一层的节点后，将这一层的元素添加进结果数组
>     res.push(curLevelNode)
>   }
>   return res
> };
> ```

### _**前序与中序遍历构造二叉树**_

> _**问题描述**_
>
> > [!important] **[LeetCode - 从前序与中序遍历序列构造二叉树](https://leetcode.cn/problems/construct-binary-tree-from-preorder-and-inorder-traversal/description/?envType=study-plan-v2&envId=top-100-liked)**
>
> - 给定两个整数数组 `preorder` 和 `inorder` ，其中 `preorder` 是二叉树的**先序遍历**， `inorder` 是同一棵树的**中序遍历**，请构造二叉树并返回其根节点。

> _**使用递归求解**_
>
> - 使用先序遍历的第一个元素确定根节点。然后在中序遍历中找到根节点，这将中序遍历分为左子树和右子树。使用中序遍历的这个分割点，可以确定先序遍历中左子树和右子树的节点。递归地对左子树和右子树重复这一过程，直到所有节点都被处理。
> 
> - 时间复杂度：$O(n)$，其中 _**n**_ 是树中的节点个数。
> 
> - 空间复杂度：$O(n)$，除去返回的答案需要的 O(n) 空间之外，我们还需要使用 O(n) 的空间存储哈希映射，以及 O(h)（其中 h 是树的高度）的空间表示递归时栈空间。这里 h<n，所以总空间复杂度为 O(n)。
> 
> ```Java
> var buildTree = function(preorder, inorder) {
>   // 使用先序遍历的第一个元素确定根节点
>   // 在中序遍历中找到根节点，这将中序遍历分为两部分：左子树和右子树
>   // 使用中序遍历的这个分割点，可以确定先序遍历中左子树和右子树的节点
>   // 递归地对左子树和右子树重复这一过程，直到所有节点都被处理
>   if (!preorder.length || !inorder.length) return null
>   // 先序遍历的第一个元素就是根节点的值
>   let rootValue = preorder[0]
>   // 根据值创建节点
>   let root = new TreeNode(rootValue)
>   // 通过 inorder 数组找到根节点的索引，记为 midIndex
>   // midIndex会将 preorder 分成两部分 => [1, midIndex]是左子树，(midIndex, preorder.length-1]是右子树
>   // midIndex会将 inorder 分成两部分 => [0, midIndex)是左子树，(midIndex, inoder.length-1]是右子树
>   let midIndex = inorder.indexOf(rootValue)
>   // 递归左子树
>   // slice() 方法返回一个由 start 和 end 决定的原数组的浅拷贝（包括 start，不包括 end）
>   root.left = buildTree(preorder.slice(1, midIndex + 1), inorder.slice(0, midIndex))
>   // 递归右子树
>   // root.right = buildTree(preorder.slice(midIndex + 1, inorder.length), inorder.slice(midIndex + 1, inorder.length))
>   root.right = buildTree(preorder.slice(midIndex + 1), inorder.slice(midIndex + 1))
>   return root
> };
> ```
