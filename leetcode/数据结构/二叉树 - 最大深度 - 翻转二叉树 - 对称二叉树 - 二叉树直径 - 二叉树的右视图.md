### _最大深度_

> _**问题描述**_
>
> > [!important] **[LeetCode - 最大深度](https://leetcode.cn/problems/maximum-depth-of-binary-tree/description/?envType=study-plan-v2&envId=top-100-liked)**
>
> - 给定一个二叉树 `root` ，返回其最大深度。二叉树的 **最大深度** 是指从根节点到最远叶子节点的最长路径上的节点数。

> _**解法 1 ⇒ 深度优先搜索 & 递归**_
>
> - 知道左子树和右子树的最大深度 **l** 和 **r**，则该二叉树的最大深度即为`**max(l, r) + 1**`
> 
> - 时间复杂度：$O(n)$，其中 _**n**_ 为二叉树节点的个数。每个节点在递归中只被遍历一次
> 
> - 空间复杂度：$O(height)$，其中 _**height**_ 表示二叉树的高度。递归函数需要栈空间，而栈空间取决于递归的深度，则空间复杂度等价于二叉树的高度
> 
> ```Java
> 
> var maxDepth = function(root) {
>   const afterOrder = (root) => {
>     if (!root) return 0
>     let left = afterOrder(root.left)
>     let right = afterOrder(root.right)
>     return 1 + Math.max(left, right)
>   }
>   let max = afterOrder(root)
>   return max
> };
> ```

> _**解法 2 ⇒ 广度优先搜索 & 队列**_
>
> > [!important] **[廖雪峰 - 使用 Queue](https://www.liaoxuefeng.com/wiki/1252599548343744/1265121791832960)**
> >
> > **[Oracle - Interface Queue
> >
> > [菜鸟教程 - 队列（Queue）用法](https://www.runoob.com/java/data-queue.html)**
>
> - LinkedList 类实现 Queue 接口，因此我们可以把 LinkedList 当成 Queue 来用
> 
> - 时间复杂度：$O(n)$，其中 _**n**_ 为二叉树的节点个数。与方法一同样的分析，每个节点只会被访问一次。
> 
> - 空间复杂度：此方法空间的消耗取决于队列存储的元素数量，其在最坏情况下会达到 $O(n)$
> 
> ```Java
> public int maxDepth(TreeNode root) {
>     if (root == null) return 0;
>     // 使用 LinkedList 实现队列
>     Queue<TreeNode> queue = new LinkedList<>();
>     // 将 root 添加到队列
>     queue.offer(root);
>     // 记录深度
>     int ans = 0;
>     while (!queue.isEmpty()) {
>         int size = queue.size();
>         while (size > 0) {
>             // 取队首元素并删除
>             TreeNode node = queue.poll();
>             if (node.left != null) queue.offer(node.left);
>             if (node.right != null) queue.offer(node.right);
>             //  size 变量反映的是当前层的节点数，不会随着 queue.offer() 操作而增加
>             size--;
>         }
>         ans++;
>     }
>     return ans;
> }
> ```

### _**翻转二叉树**_

> _**问题描述**_
>
> > [!important] **[LeetCode - 翻转二叉树](https://leetcode.cn/problems/invert-binary-tree/description/?envType=study-plan-v2&envId=top-100-liked)**
>
> - 给你一棵二叉树的根节点 `root` ，翻转这棵二叉树，并返回其根节点。>

> _**解法 1 ⇒ 递归**_
>
> - 时间复杂度：$O(N)$，其中 _**N**_ 为二叉树节点的数目。我们会遍历二叉树中的每一个节点，对每个节点而言，我们在常数时间内交换其两棵子树。
> 
> - 空间复杂度：$O(N)$。使用的空间由递归栈的深度决定，它等于当前节点在二叉树中的高度。在平均情况下，二叉树的高度与节点个数为对数关系，即 $O(log⁡N)$。而在最坏情况下，树形成链状，空间复杂度为 $O(N)$。
> 
> ```Java
> 
> var invertTree = function(root) {
>   if (!root) return null
>   // 后续遍历
>   let left = invertTree(root.left)
>   let right = invertTree(root.right)
>   // 交换左右子树
>   root.right = left
>   root.left = right
>   return root
> };
> ```

> _**解法 2 ⇒ 队列**_
>
> - 时间复杂度：$O(n)$。同样每个节点都需要入队列/出队列一次
> 
> - 空间复杂度：$O(n)$。最坏情况下包含所有的叶子节点，完全二叉树叶子节点是 **n/2** 个。
> 
> ```Java
> public TreeNode invertTree(TreeNode root) {
>     if (root == null) return null;
>     // 使用 LinkedList 实现队列
>     Queue<TreeNode> queue = new LinkedList<>();
>     queue.offer(root); // 添加元素
>     while (!queue.isEmpty()) {
>         // 每次都从队列中拿一个节点，并交换这个节点的左右子树
>         TreeNode tmp = queue.poll();
>         // 交换这个节点的左右子树
>         TreeNode left = tmp.left;
>         tmp.left = tmp.right;
>         tmp.right = left;
>         if (tmp.left != null) queue.offer(tmp.left);
>         if (tmp.right != null) queue.offer(tmp.right);
>     }
>     return root;
> }
> ```

### _对称二叉树_

> _**问题描述**_
>
> > [!important] **[LeetCode - 对称二叉树](https://leetcode.cn/problems/symmetric-tree/description/?envType=study-plan-v2&envId=top-100-liked)**
>
> - 给你一个二叉树的根节点 `root` ， 检查它是否轴对称。

> _**使用递归**_
>
> - 对于树中 **任意两个对称节点** `L` 和 `R` ，一定有
> 
> - `L.val = R.val`：即此两对称节点值相等。
> 
> - `L.left.val = R.right.val` ：即 L 的 左子节点 和 R 的 右子节点 对称。
> 
> - `L.right.val = R.left.val` ：即 L 的 右子节点 和 R 的 左子节点 对称。
> 
> ```Java
> /**
>  * Definition for a binary tree node.
>  * function TreeNode(val, left, right) {
>  *     this.val = (val===undefined ? 0 : val)
>  *     this.left = (left===undefined ? null : left)
>  *     this.right = (right===undefined ? null : right)
>  * }
>  */
> /**
>  * @param {TreeNode} root
>  * @return {boolean}
>  */
> var isSymmetric = function(root) {
>   if (!root) return true
> 
>   // 任意两个对称节点 L 和 R ，一定有
>   // L 和 R 都为空时，L 和 R 对称
>   // L.val = R.val：即此两对称节点值相等。
>   // L.left.val = R.right.val ：即 L 的 左子节点 和 R 的 右子节点 对称。
>   // L.right.val = R.left.val：即 L 的 右子节点 和 R 的 左子节点 对称。
>   const check = (L, R) => {
>     // L 和 R 都为空时，L 和 R 对称
>     if (!L && !R) return true
>     // L 和 R 中有一个不存在 或 两个值不等，则返回 false
>     if (!L || !R || L.val !== R.val) return false
>     // 当 L 和 R 的值相等，且都不为空时，进入递归
>     return check(L.left, R.right) && check(L.right, R.left)
>   }
> 
>   return check(root.left, root.right)
> };
> ```

### _**二叉树的直径**_

> _**问题描述**_
>
> > [!important] **[LeetCode - 二叉树的直径](https://leetcode.cn/problems/diameter-of-binary-tree/description/?envType=study-plan-v2&envId=top-100-liked)**
>
> - 给你一棵二叉树的根节点，返回该树的 **直径** 。二叉树的 **直径** 是指树中任意两个节点之间最长路径的 **长度** 。这条路径可能经过也可能不经过根节点 `root` 。两节点之间路径的 **长度** 由它们之间边数表示。

> _**使用递归**_
>
> - `**二叉树的直径 = 左子树的最大深度 + 右子树的最大深度**`
> 
> ```Java
> var diameterOfBinaryTree = function(root) {
>   // 二叉树的直径 = 左子树的最大深度 + 右子树的最大深度
>   let res = 0
>   // 递归
>   const maxDepth = (root) => {
>     if (!root) return 0
>     let left = maxDepth(root.left)
>     let right = maxDepth(root.right)
>     res = Math.max(left + right, res)
>     return 1 + Math.max(left, right)
>   }
>   maxDepth(root)
>   return res
> };
> ```

### _**二叉树的右视图**_

> _**问题描述**_
>
> > [!important]
> >
> > **[LeetCode - 二叉树的右视图](https://leetcode.cn/problems/binary-tree-right-side-view/description/?envType=study-plan-v2&envId=top-100-liked)**
>
> - 给定一个二叉树的 **根节点** `root`，想象自己站在它的右侧，按照从顶部到底部的顺序，返回从右侧所能看到的节点值。

> _**使用队列**_
>
> ```Java
> var rightSideView = function(root) {
>   // 使用队列遍历二叉树
>   if (!root) return []
>   let queue = [root]
>   // 声明返回结果
>   let res = []
>   while (queue.length) {
>     // 用一个变量记录本层的节点数
>     let nodeSize = queue.length
>     for (let i = 0; i < nodeSize; i++) {
>       // 将当前节点出队
>       let node = queue.shift()
>       // 如果 i === nodeSize-1，说明遍历到了本层最后一个元素
>       if (i === nodeSize - 1) res.push(node.val)
>       // 将当前节点的左右孩子入队
>       if (node.left) queue.push(node.left)
>       if (node.right) queue.push(node.right)
>     }
>   }
>   return res
> };  
> ```
