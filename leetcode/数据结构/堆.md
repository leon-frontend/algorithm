> **数组存储**
>
> - 完全二叉树 非常适合用数组来表示
> 
> - 由于 堆正是一种完全二叉树，因此我们将采用 数组 来存储堆

> _**数组中索引的特点**_
>
> ```JavaScript
> /* 获取左子节点的索引 */

> }
> 
> /* 获取右子节点的索引 */
> \#right(i) {
>     return 2 * i + 2;
> }
> 
> /* 获取父节点的索引 */
> \#parent(i) {
>     return Math.floor((i - 1) / 2); // 向下整除

# 2. 元素入堆 / 出堆

> _**入堆**_
>
> - 给定元素 `**val**` ，我们首先将其添加到堆底
> 
> - 添加之后，由于 `**val**` 可能大于堆中其他元素，堆的成立条件可能已被破坏，因此需要修复从插入节点到根节点的路径上的各个节点，这个操作被称为「堆化 _**heapify**_」
> 
> - _==**从底至顶 执行堆化，直至 越过根节点 或 遇到无须交换的节点 时结束**==_
>     
>     - 我们比较 插入节点 与 其父节点 的值，如果插入节点更大，则将它们交换
>     

> _**出堆**_
>
> - 交换 堆顶元素 与 堆底元素（交换根节点与最右叶节点）
> 
> - 交换完成后，将 堆底 从列表中删除（即 删除堆顶元素）
> 
> - _==**从根节点开始，从顶至底 执行堆化**==_

# 3. 建堆

> _**通过遍历堆化实现**_
>
> - 将列表所有元素原封不动地添加到堆中，此时堆的性质尚未得到满足
> 
> - 倒序遍历堆（层序遍历的倒序），依次对每个 非叶节点 执行“从顶至底堆化”
>     
>     - 由于叶节点没有子节点，因此它们天然就是合法的子堆，无须堆化
>     
> 
> - 每当堆化一个节点后，以该节点为根节点的子树就形成一个合法的子堆。
>     
>     - 而由于是倒序遍历，因此堆是“自下而上”构建的
>     

> _**代码示例**_
>
> ```JavaScript
> function swap(arr, i, j) {
>     [arr[i], arr[j]] = [arr[j], arr[i]];
> }
> 
> // 小顶堆的堆化
> function heapify(arr, n, i) {
>     let smallest = i;
>     let l = 2 * i + 1;
>     let r = 2 * i + 2;
> 
>     if (l < n && arr[l] < arr[smallest]) {
>         smallest = l;
>     }
> 
>     if (r < n && arr[r] < arr[smallest]) {
>         smallest = r;
>     }
> 
>     if (smallest !== i) {
>         swap(arr, i, smallest);
>         heapify(arr, n, smallest); // 至顶向底堆化
>     }
> }
> 
> // 建立小顶堆
> function buildMinHeap(arr, n) {
> 		// 最后一个非叶节点是最后一个节点的父节点
>     let startIdx = Math.floor(n / 2) - 1;
> 		
> 		// 倒序遍历堆（层序遍历的倒序），依次对每个 非叶节点 执行“从顶至底堆化”
>     for (let i = startIdx; i >= 0; i--) {
>         heapify(arr, n, i);
>     }
> }
> ```

# 4. Top-k 问题

> _**问题描述**_
>
> > [!important] ** [Hello 算法 - Top-k 问题](https://www.hello-algo.com/chapter_heap/top_k/) **
>
> - 给定一个长度为 _**n**_ 的 无序 数组 _**nums**_，请返回数组中最大的 _**k**_ 个元素

> _**使用堆 O(nlogk)**_
>
> - 初始化一个小顶堆，其 堆顶元素最小（小顶堆）
> 
> - 先将数组的 前 _**k**_ 个元素依次入堆，建堆操作
> 
> - 从第 _**k+1**_ 个元素开始，若当前元素大于堆顶元素，则将堆顶元素出堆，并将当前元素入堆
> 
> - 遍历完成后，堆中保存的就是最大的 _**k**_ 个元素
> 
> ```JavaScript
> function swap(arr, i, j) {
>     [arr[i], arr[j]] = [arr[j], arr[i]]
> }
> 
> // 小顶堆的堆化
> // arr 表示数组；n 表示数组长度；i 表示从下标 i 开始堆化
> function heapify(arr, n, i) {
>     let smallest = i
>     let l = 2 * i + 1
>     let r = 2 * i + 2
> 
>     if (l < n && arr[l] < arr[smallest]) smallest = l
>     if (r < n && arr[r] < arr[smallest]) smallest = r
> 
>     if (smallest !== i) {
>         swap(arr, i, smallest)
>         heapify(arr, n, smallest) // 至顶向底堆化
>     }
> }
> 
> // 建立小顶堆
> function buildMinHeap(arr, n) {
> 		// 最后一个非叶节点是最后一个节点的父节点
>     let startIdx = Math.floor(n / 2) - 1
> 		
> 		// 倒序遍历堆（层序遍历的倒序），依次对每个 非叶节点 执行“从底至顶堆化”
>     for (let i = startIdx; i >= 0; i--) heapify(arr, n, i)
> }
> 
> function findKLargest(nums, k) {
>     let minHeap = nums.slice(0, k) // 获取数组中的前 K 个元素
>     buildMinHeap(minHeap, k) // 将数组中的前 K 个元素建堆
> 
>     for (let i = k; i < nums.length; i++) {
>         if (nums[i] > minHeap[0]) {
>             minHeap[0] = nums[i]
>             heapify(minHeap, k, 0) // 至顶向底堆化
>         }
>     }
>     return minHeap[0]
> }
> 
> return findK(nums, k)
> ```

# 5. 前 K 个高频元素

> _**问题描述**_
>
> > [!important] ** [LeetCode - 前 K 个高频元素](https://leetcode.cn/problems/top-k-frequent-elements/description/?envType=study-plan-v2&envId=top-100-liked) **
>
> - 给你一个整数数组 `nums` 和一个整数 `k` ，请你返回其中出现频率前 `k` 高的元素。
> 
> - 你可以按 **任意顺序** 返回答案。

> _**堆 + 对象映射**_
>
> ```JavaScript
> /**
>  * @param {number[]} nums
>  * @param {number} k
>  * @return {number[]}
>  */
> var topKFrequent = function(nums, k) {
>   // 创建一个对象，记录元素和出现次数的对应关系
>   const frequency = {}
>   // 遍历每个元素，更新 frequency
>   nums.forEach(num => {
>     // 若元素已存在于 frequency 中，则出现的次数加 1；否则新增该元素
>     if (frequency[num]) frequency[num]++
>     else frequency[num] = 1
>   })
>   // 通过 frequency 得到不重复的数组
>   const keysArr = Object.keys(frequency)
> 
>   // -----------------------------------------------------
>   // 小顶堆的堆化
>   const heapify = (arr, len, index) => {
>     let smallest = index
>     const left = index * 2 + 1, right = index * 2 + 2
> 
>     if (left < len && frequency[arr[left]] < frequency[arr[smallest]]) smallest = left
>     if (right < len && frequency[arr[right]] < frequency[arr[smallest]]) smallest = right
> 
>     if (smallest !== index) {
>       [arr[smallest], arr[index]] = [arr[index], arr[smallest]]
>       heapify(arr, len, smallest)  // 至顶向底堆化
>     }
>   }
> 
>   // 建立小顶堆
>   const buildMinHeap = (arr, len) => {
>     const startIndex = Math.floor(len / 2) - 1
>     for (let i = startIndex; i >= 0; i--) heapify(arr, len, i)
>   }
> 
>   // 找出前 k 个高频元素
>   // 从不重复的数组里，浅拷贝前 k 个元素
>   const minHeap = keysArr.slice(0, k)
>   // 对前 k 个元素建堆
>   buildMinHeap(minHeap, k)
> 
>   // 从第 k+1 个元素开始入堆
>   for (let i = k; i < keysArr.length; i++) {
>     if (frequency[keysArr[i]] > frequency[minHeap[0]]) {
>       minHeap[0] = keysArr[i]
>       heapify(minHeap, k, 0) // 至顶向底堆化
>     }
>   }
> 
>   return minHeap
> };
> ```

# 6. 柱状图中最大的矩形

> _**问题描述**_
>
> > [!important]  [LeetCode - 柱状图中最大的矩形](https://leetcode.cn/problems/largest-rectangle-in-histogram/description/?envType=study-plan-v2&envId=top-100-liked) 
>
> - 给定 _**n**_ 个非负整数，用来表示柱状图中各个柱子的高度
> 
> - 每个柱子彼此相邻，且宽度为 1
> 
> - 求在该柱状图中，能够勾勒出来的矩形的最大面积

> _**代码详解**_
>
> > [!important] ** [LeetCode - 题解](https://leetcode.cn/problems/largest-rectangle-in-histogram/solutions/142012/bao-li-jie-fa-zhan-by-liweiwei1419/?envType=study-plan-v2&envId=top-100-liked) **
>
> ```JavaScript
> /**
>  * @param {number[]} heights
>  * @return {number}
>  */
> 
> // 从左到右遍历数组，从右到左确定最大面积 => 栈
> 
> // 在找宽度时，对于当前栈顶元素，i 为右边界，
> // 左边界是栈中第一个比当前栈顶元素的小的元素 => 即当前栈顶元素出栈后的栈顶元素
> // 在题解中查看选 index = 4 的情况，来理解左右边界的选择
> 
> var largestRectangleArea = function(heights) {
>   // 维护一个递增的栈，记录递增序列的索引
>   let stack = []
>   // 在数组前后各添加一个高度为0的柱子，方便处理边界情况
>   heights = [0, ...heights, 0] 
>   let maxArea = 0;
>   
>   for (let i = 0; i < heights.length; i++) {
>     // 当前元素小于栈顶元素时，将栈顶元素出栈，并记录高度和宽度
>     // 需要进行循环比较，因为当前元素可能小于多个栈顶元素
>     while (stack.length && heights[i] < heights[stack[stack.length - 1]]) {
>       // 计算矩形高度，并将栈顶元素出栈
>       let height = heights[stack.pop()]
>       // 计算矩形宽度 => 看上面的注释
>       let width = i - stack[stack.length - 1] - 1
>       maxArea = Math.max(maxArea, height * width)
>     }
> 
>     // 将每个元素的索引都入栈
>     stack.push(i)
>   }
> 
>   return maxArea;
> };
> ```
