### _**有效的括号**_

> _**题目描述**_
>
> > [!important] **[LeetCode - 有效的括号](https://leetcode.cn/problems/valid-parentheses/description/?envType=study-plan-v2&envId=top-100-liked)**
>
> - 给定一个只包括 '('，')'，'{'，'}'，'['，']' 的字符串 _**s**_ ，判断字符串是否有效。
> 
> - 有效字符串需满足：1. 左括号必须用相同类型的右括号闭合。
> 
> - 左括号必须以正确的顺序闭合
> 
> - 每个右括号都有一个对应的相同类型的左括号
> 
> ![[栈 - 1.png|Untitled 72.png]]
>
> ```JavaScript
> /**
>  * @param {string} s
>  * @return {boolean}
>  */
> var isValid = function(s) {
>   const stack = []
>   let counterbracket
>   const brackets = {
>     '(': ')',
>     '[': ']',
>     '{': '}'
>   }
> 
>   for (let char of s) {
>     if (char in brackets){
>       // 如果是 括号，则入栈
>       stack.push(char)
>     }else {
>       // 如果是反括号，则出栈，并得到相应的反括号
>       counterbracket = brackets[stack.pop()]
>       // 将当前遍历的反括号和出栈的括号进行匹配
>       if (char !== counterbracket) return false
>     }
>   }
> 
> 	// 不用 return true 是为了防止只有一个 "[" 的情况
>   return stack.length === 0
> };
> ```

### _**最小栈**_

> _**题目描述**_
>
> > [!important] **[LeetCode - 最小栈](https://leetcode.cn/problems/min-stack/description/?envType=study-plan-v2&envId=top-100-liked)**
>
> - 设计一个支持 `push` ，`pop` ，`top` 操作，并能在常数时间内检索到最小元素的栈
> 
> - 实现 `MinStack` 类: _**1.**_ `MinStack()` 初始化堆栈对象。_**2.**_ `void push(int val)` 将元素val推入堆栈。_**3.**_ `void pop()` 删除堆栈顶部的元素。_**4.**_ `int top()` 获取堆栈顶部的元素。_**5.**_ `int getMin()` 获取堆栈中的最小元素
> 
> ![[栈 - 2.png|Untitled 1 45.png]]

> _**使用一个辅助栈，与元素栈****同步插入与删除****，用于存储** **与每个元素对应的最小值**_
>
> - 当一个元素要入栈时，我们取当前辅助栈的栈顶存储的最小值，与当前元素比较得出最小值，将这个最小值插入辅助栈中
> 
> - 当一个元素要出栈时，我们把辅助栈的栈顶元素也一并弹出
> 
> - 在任意一个时刻，栈内元素的最小值就存储在辅助栈的栈顶元素中
> 
> ![[栈 - 3.png|Untitled 2 41.png]]
>
> ![[栈 - 4.png|Untitled 3 30.png]]
>
> ```JavaScript
> var MinStack = function() {
>   // 维护两个栈
>   // 1. 用于存储 val 值的栈
>   this.stack = []
>   // 2. 只存储 最小值 的栈
>   this.minStack = [Infinity]
> };
> 
> /** 
>  * @param {number} val
>  * @return {void}
>  */
> MinStack.prototype.push = function(val) {
>   // 将 val 入 stack 栈
>   this.stack.push(val)
>   // 将 最小值 入 minStack 栈 
>   this.minStack.push(Math.min(this.minStack[this.minStack.length - 1], val))
> };
> 
> /**
>  * @return {void}
>  */
> MinStack.prototype.pop = function() {
>   this.stack.pop()
>   this.minStack.pop()
> };
> 
> /**
>  * @return {number}
>  */
> MinStack.prototype.top = function() {
>   return this.stack[this.stack.length - 1]
> };
> 
> /**
>  * @return {number}
>  */
> MinStack.prototype.getMin = function() {
>   return this.minStack[this.minStack.length - 1]
> };
> 
> /**
>  * Your MinStack object will be instantiated and called as such:
>  * var obj = new MinStack()
>  * obj.push(val)
>  * obj.pop()
>  * var param_3 = obj.top()
>  * var param_4 = obj.getMin()
>  */
> ```

### _**字符串解码**_

> _**问题描述**_
>
> > [!important] **[LeetCode - 字符串解码](https://leetcode.cn/problems/decode-string/description/?envType=study-plan-v2&envId=top-100-liked)**
>
> - 给定一个经过编码的字符串，返回它解码后的字符串。
> 
> - 编码规则为:`k[encoded_string]`，表示其中方括号内部的`encoded_string`正好重复 `k` 次。注意 `k` 保证为正整数。
> 
> - 你可以认为输入字符串总是有效的；输入字符串中没有额外的空格，且输入的方括号总是符合格式要求的。
> 
> - 此外，你可以认为原始数据不包含数字，所有的数字只表示重复的次数 `k` ，例如不会出现像 `3a` 或 `2[4]` 的输入。 

> _**使用两个辅助栈**_
>
> - 创建两个栈，一个用于保存数字（即重复的次数），另一个用于保存字符串。然后遍历给定的编码字符串。
> 
> - 当遇到数字时，解析出完整的数字（==可能包含多位==），并将其压入数字栈
> 
> - 当遇到左方括号`**[**`时，将当前的字符串（如果有）压入字符串栈，数字压入数字栈==，====并重置当前字符串和重复的次数====。==重置是为了防止嵌套的 字符串和次数 发生混淆
> 
> - 当遇到右方括号`**]**`时，表示一个编码字符串结束。出栈，此时 **currentStr** 中记录了 ==`**[]**`== 中的字符串。此时，从数字栈中弹出一个数字，表示重复次数。==将当前字符串重复相应的次数，然后与字符串栈顶的字符串相连。==
> 
> - 如果遇到 字母，将其添加到当前字符串中。因为==一个字符串可能有一个或多个字符。==
> 
> - 继续遍历直到字符串结束，最后返回最终构建的字符串
> 
> ```JavaScript
> /**
>  * @param {string} s
>  * @return {string}
>  */
> var decodeString = function(s) {
>   const numStack = [] // 存储重复次数的栈
>   const strStack = [] // 存储字符串的栈
> 
>   // 初始化 重复次数 和 当前字符串
>   let multi = 0, currentStr = ""
> 
>   // 遍历字符串 s
>   for (let char of s) {
>     // 如果是数字，则将字符型的数字转换为数字型，数字可能是多位数
>     // 使用 isNaN 方法判断字符是否是数字 => isNaN(value)
>     // 如果给定值在被转换为数字后为 NaN 则返回值为 true；否则为 false
>     if (!isNaN(char)) {
>       // 由于数字可能是多位数，所以要累加
>       multi = multi * 10 + Number(char)
>     }else if (char === '['){ // 如果是左括号
>       // 将 当前字符串 压入字符串栈，将 当前数字 压入数字栈
>       numStack.push(multi)
>       strStack.push(currentStr)
>       // 重置 当前字符串 和 重复的次数
>       // 重置是为了防止嵌套的 字符串和次数 发生混淆
>       multi = 0
>       currentStr = ""
>     }else if (char === ']'){ // 如果是右括号
>       // 出栈，此时 currentStr 中记录了 [] 中的字符串
>       // numStack 出栈
>       let repeatTimes = numStack.pop()
>       // strStack 出栈，并连接当前字符串
>       currentStr = strStack.pop() + currentStr.repeat(repeatTimes)
>     }else { // 如果是字符
>       // 可能有多个连续字符，加入当前字符串
>       currentStr += char
>     }
>   }
> 
>   return currentStr
> };
> ```

### _**每日温度**_

> _**问题描述**_
>
> > [!important] **[LeetCode - 每日温度](https://leetcode.cn/problems/daily-temperatures/description/?envType=study-plan-v2&envId=top-100-liked)**
>
> - 给定一个整数数组 `temperatures` ，表示每天的温度，返回一个数组 `answer` ，其中 `answer[i]` 是指对于第 `i` 天，下一个更高温度出现在几天后。如果气温在这之后都不会升高，请在该位置用 `0` 来代替。
>
> ```JavaScript
> /**
>  * @param {number[]} temperatures
>  * @return {number[]}
>  */
> var dailyTemperatures = function(temperatures) {
>   // 定义存储 temperature 数组索引的栈
>   const indexStack = []
> 
>   // 定义结果数组，最大值索引及其后面的索引都是 0
>   const res = Array(temperatures.length).fill(0)
> 
>   // 先将 temperature 数组的第一个索引入栈
>   indexStack.push(0)
> 
>   // 遍历 temperature 数组
>   for (let index = 1; index < temperatures.length; index++) {
>     // 比较 当前遍历到的 temperature 数组元素 和 indexStack 的栈顶元素
>     // 如果当前遍历的元素大于栈顶元素，则栈顶元素出栈，并更新结果数组，然后将当前遍历元素 索引 入栈
>     // 有可能当前遍历元素大于多个栈中的元素，所以要用循环
>     // indexStack.length 判断数组是否为空
>     while (indexStack.length && temperatures[index] > 
> 													temperatures[indexStack[indexStack.length - 1]]){
>       // 出栈
>       const topIndex = indexStack.pop()
>       // 合并 出栈 和 更新结果数组 操作
>       res[topIndex] = index - topIndex
>     }
>     // 如果当前遍历的元素 小于等于 栈顶元素，则将当前遍历元素的索引入栈
>     indexStack.push(index)
>   }
> 
>   return res
> };
> ```
