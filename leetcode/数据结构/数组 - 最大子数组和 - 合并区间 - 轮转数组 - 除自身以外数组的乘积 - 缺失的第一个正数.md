### _**最大子数组和**_

> _**问题描述**_
>
> > [!important] **[LeetCode - 最大子数组和](https://leetcode.cn/problems/maximum-subarray/description/?envType=study-plan-v2&envId=top-100-liked)**
>
> - 给你一个整数数组 `nums` ，请你找出一个具有最大和的连续子数组（子数组最少包含一个元素），返回其最大和。子数组是数组中的一个连续部分。 

> _**使用动态规划**_
>
> - 该问题的状态转移方程（以下两种方程等价）
>     
>     ![[数组 - 2.png|Untitled 1 40.png]]
>     
>     ![[数组 - 3.png|Untitled 2 36.png]]
>     
> 
> - _==**dp[i] 表示以 nums[i] 结尾的最大子数组和**==_
> 
> ```Java
> /**
>  * @param {number[]} nums
>  * @return {number}
>  */
> var maxSubArray = function(nums) {
>   // 声明返回结果
>   let max = nums[0]
>   // 初始化dp表
>   // dp[i] 表示以 nums[i] 结尾的最大子数组和
>   const dp = Array(nums.length)
>   // 边界条件
>   dp[0] = nums[0]
>   // 状态转移方程
>   for (let i = 1; i < nums.length; i++) {
>     dp[i] = Math.max(dp[i-1] + nums[i], nums[i])
>     max = Math.max(max, dp[i])
>   }
>   return max
> };
> ```

### _**合并区间**_

> _**问题描述**_
>
> > [!important] **[LeetCode - 合并区间](https://leetcode.cn/problems/merge-intervals/description/?envType=study-plan-v2&envId=top-100-liked)**
>
> - 以数组 `intervals` 表示若干个区间的集合，其中单个区间为 `intervals[i] = [starti, endi]` 。请你合并所有重叠的区间，并返回 _一个不重叠的区间数组，该数组需恰好覆盖输入中的所有区间_ 。 

> _**排序**_
>
> - 在排完序的列表中，可以合并的区间一定是连续的
> 
> ```Java
> /**
>  * @param {number[][]} intervals
>  * @return {number[][]}
>  */
> var merge = function(intervals) {
>   // 声明返回结果
>   const res = []
>   // 将 intervals 数组排序 => 第一列升序
>   // 在排完序的列表中，可以合并的区间一定是连续的
>   intervals.sort((arr1, arr2) => arr1[0] - arr2[0])
>   // 遍历 intervals 的每一行
>   for (let row = 0; row < intervals.length; row++) {
>     // 保存当前区间的两个值
>     let L = intervals[row][0], R = intervals[row][1] 
>     // 将 当前区间的左值 和 res 中最新区间的右值 比较
>     // 若 当前区间的左值 > res 中最新区间的右值 => 则将当前区间加入 res 数组
>     // 且当 res 为空时，将当前区间加入 res 数组
>     if (res.length === 0 || res[res.length - 1][1] < L) res.push(intervals[row])
>     // 否则，它们重合 => 需要用当前区间的右端点更新数组 ans 中最后一个区间的右端点，将其置为二者的较大值
>     else res[res.length - 1][1] = Math.max(R, res[res.length - 1][1])
>   }
>   return res
> };
> ```

### _**轮转数组**_

> _**问题描述**_
>
> > [!important] **[LeetCode - 轮转数组](https://leetcode.cn/problems/rotate-array/description/?envType=study-plan-v2&envId=top-100-liked)**
>
> - 给定一个整数数组 `nums`，将数组中的元素向右轮转 `k` 个位置，其中 `k` 是非负数。 
>
> ```JavaScript
> /**
>  * @param {number[]} nums
>  * @param {number} k
>  * @return {void} Do not return anything, modify nums in-place instead.
>  */
> var rotate = function(nums, k) {
>   // 反转数组函数
>   const reverse = (nums, left, right) => {
>     while (left < right) {
>       [nums[left], nums[right]] = [nums[right], nums[left]]
>       left++
>       right--
>     }
>   }
>   // 对 k 取余，避免 k > nums.length => 比如 nums=[1,2] k=3 => nums=[2,1] 
>   k %= nums.length
>   reverse(nums, 0, nums.length - 1)
>   reverse(nums, 0, k - 1)
>   reverse(nums, k, nums.length - 1)
>   return nums
> };
> ```

### _**除自身以外数组的乘积**_

> _**问题描述**_
>
> > [!important] **[LeetCode - 除自身以外数组的乘积](https://leetcode.cn/problems/product-of-array-except-self/description/?envType=study-plan-v2&envId=top-100-liked)**
>
> - 给你一个整数数组 **nums**，返回数组 **answer** ，其中`answer[i]`等于`nums`中除`nums[i]`之外其余各元素的乘积 。题目数据保证数组`nums`之中任意元素的全部前缀元素和后缀的乘积都在  **32 位** 整数范围内。请**不要使用除法，**且在 _`O(n)`_ 时间复杂度内完成此题。

> _**动态规划 + 两个数组**_
>
> - 使用两个数组分别记录当前元素左边的乘积和右边的乘积
> 
> - 时间复杂度：$O(N)$；空间复杂度：$O(N)$
> 
> ```Java
> /**
>  * @param {number[]} nums
>  * @return {number[]}
>  */
> var productExceptSelf = function(nums) {
>   const res = []
>   // 创建两个数组，分别记录从左往右的乘积和从右往左的乘积
>   const left = Array(nums.length), right = Array(nums.length)
>   // 初始化两个数组
>   left[0] = right[nums.length - 1] = 1
>   // 从左往右记录乘积
>   for (let i = 1; i < nums.length; i++) left[i] = left[i - 1] * nums[i - 1]
>   // 从右往左记录乘积
>   for (let i = nums.length - 2; i >= 0; i--) right[i] = right[i + 1] * nums[i + 1]
>   for (let i = 0; i < nums.length; i++) {
>     // left[i] 表示 i 的左前缀乘积，right[i] 表示 i 的右前缀乘积
>     res[i] = left[i] * right[i]
>   }
>   return res
> };
> ```

### _**缺失的第一个正数**_

> _**问题描述**_
>
> > [!important] **[LeetCode - 缺失的第一个正数](https://leetcode.cn/problems/first-missing-positive/description/?envType=study-plan-v2&envId=top-100-liked)**
>
> - 给你一个未排序的整数数组 `nums` ，请你找出其中没有出现的最小的正整数。请你实现时间复杂度为 **O(n)**，并且只使用常数级别额外空间的解决方案。 

> _**置换算法**_
>
> 1. 遍历数组，将所有正整数放置到正确的位置上。即，如果数字 `**x**` 在范围 `**[1, n]**` 内，那么将它放在索引 `**x - 1**` 的位置上。
> 
> 1. 再次遍历数组，找到第一个位置 `**i**`，使得 `**nums[i] != i + 1**`。那么缺失的最小正整数就是 `**i + 1**`。
> 
> 1. 如果遍历完数组仍然没有找到缺失的正整数，则缺失的正整数为 `**n + 1**`。
> 
> ```JavaScript
> /**
>  * @param {number[]} nums
>  * @return {number}
>  */
> var firstMissingPositive = function(nums) {
>   // 采用置换的方法，若 1 <= 数字x <=n，那么将其放在索引 x-1 的位置上
>   // 示例 [3, 4, -1, 1]，恢复后的数组为 [1, -1, 3, 4]，可以知道缺失的数为 2  
>   // 第一次遍历数组，将满足 1 <= 数字x <=n 的数字x 放在索引 x-1 的位置上
>   for (let i = 0; i < nums.length; i++) {
>     // nums[nums[i] - 1] 表示索引 x-1 的位置上的元素值
>     // 使用 while 循环是因为 => 换过来的值 x 也可能不在索引 x-1 的位置上
>     while (nums[i] > 0 && nums[i] <= nums.length && nums[nums[i] - 1] !== nums[i]) {
>       // 交换两个元素，使得元素 x 放在索引 x-1 的位置上
>       [nums[nums[i] - 1], nums[i]] = [nums[i], nums[nums[i] - 1]]
>     }
>   }
>   // 第二次遍历循环，找出第一个满足 nums[i] !== i+1 的 i (i 是下标)
>   // 最小缺失的整数就是 i+1
>   for (let i = 0; i < nums.length; i++) {
>     if (nums[i] !== i + 1) return i + 1
>   }
>   // 如果遍历完数组仍然没有找到缺失的正整数，则缺失的正整数为`n+1"。
>   return nums.length + 1
> };
> ```
