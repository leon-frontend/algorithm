### _问题描述_

> 给定一个包含红色、白色和蓝色、共 `n` 个元素的数组 `nums` ，**[原地](https://baike.baidu.com/item/%E5%8E%9F%E5%9C%B0%E7%AE%97%E6%B3%95)**对它们进行排序，使得相同颜色的元素相邻，并按照红色、白色、蓝色顺序排列。
>
> - 我们使用整数 `0`、 `1` 和 `2` 分别表示红色、白色和蓝色。
> 
> - 必须在不使用库内置的 sort 函数的情况下解决这个问题。

---

### _算法分析_

> _**初始化三个指针**_
>
> - _**left**_ 指向数组起始位置，_**right**_ 指向数组末尾，_**curr**_ 指向当前元素位置
> 
> - _**left**_ 指向下一个 _**0**_ 的位置，_**curr**_ 遍历所有节点，_**right**_ 指向下一个 _**2**_ 的位置
> 
> - 初始时 `**left = 0, right = n - 1, curr = 0**`

> _**当 curr 指针小于等于 right 指针时，执行循环**_
>
> - 若`**nums[curr]**`为 _**0**_，则将其与`**nums[left]**`交换，并将 _**left**_ 和 _**curr**_ 指针同时向右移动一位
> 
> - 若==`**nums[curr]**`==为 _**1**_，则 _**curr**_ 指针向右移动一位
>     
>     - 当 _**curr**_ 和 _**left**_ 不等时，_**left**_ 只可能指向 _**1 或 0**_
>     
> 
> - 若`**nums[curr]**`为 _**2**_，则将其与`**nums[right]**`交换，并将 _**right**_ 指针向左移动一位

> 当 _**curr**_ 指针大于 _**right**_ 指针时，遍历结束

---

### _代码详解_

> _**Code**_
>
> ```JavaScript
> /**
>  * @param {number[]} nums
>  * @return {void} Do not return anything, modify nums in-place instead.
>  */
> var sortColors = function(nums) {
>   // 声明三个指针
>   let left = 0, right = nums.length-1, cur = 0
> 
>   // 当 cur 指针大于 right 指针时，遍历结束
>   while (cur <= right) {
>     if (nums[cur] === 0) {
>       [nums[left], nums[cur]] = [nums[cur], nums[left]];
>       cur++
>       left++
>     }else if (nums[cur] === 1) {
>       cur++
>     }else {
>       [nums[right], nums[cur]] = [nums[cur], nums[right]];
>       right--
>     }
>   }
> };
> ```
