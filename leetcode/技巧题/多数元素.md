### _问题描述_

> 给定一个大小为 `n` 的数组 `nums` ，返回其中的多数元素。多数元素是指在数组中出现次数 **大于** `⌊ n/2 ⌋` 的元素。
>
> 你可以假设数组是非空的，并且给定的数组总是存在多数元素。

---

### _代码详解_

> _**Code**_
>
> ```JavaScript
> /**
>  * @param {number[]} nums
>  * @return {number}
>  */
> var majorityElement = function(nums) {
>   const count = Math.floor(nums.length / 2)
> 
>   // 创建对象 => 映射 元素和出现的次数
>   const map = {}
> 
>   for (let num of nums) {
>     map[num] = (map[num] || 0) + 1
>     if (map[num] > count) return num
>   } 
> };
> ```
